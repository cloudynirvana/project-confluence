# Data Directory — Project Confluence

This directory holds real metabolomics data for generator calibration.

## Directory Structure

```
data/
├── raw/                    # Original downloaded files (DO NOT modify)
│   ├── ccle_metabolomics.csv     # DepMap/CCLE metabolomics (download manually)
│   └── lung_metabolite.csv       # Kaggle lung cancer metabolomics
├── processed/              # SAEM-mapped and normalized profiles
│   └── saem_profiles.json        # Generated by data pipeline
├── download_ccle.py        # Download script for CCLE data
└── README.md               # This file
```

## How to Get Data

### Option 1: DepMap/CCLE Metabolomics (Recommended)

1. Go to [depmap.org/portal/download](https://depmap.org/portal/download/all/)
2. Search for "Metabolomics" in the datasets
3. Download the metabolomics CSV file
4. Place it in `data/raw/ccle_metabolomics.csv`

### Option 2: Kaggle Lung Cancer Metabolomics

1. Go to [kaggle.com](https://www.kaggle.com/) and search for "lung metabolite"
2. Download `lung_metabolite.csv`
3. Place in `data/raw/lung_metabolite.csv`

### Option 3: Metabolomics Workbench

1. Go to [metabolomicsworkbench.org](https://www.metabolomicsworkbench.org/)
2. Search for isotope tracing studies (13C-glucose, 13C-glutamine)
3. Download study data and place in `data/raw/`

## Running the Pipeline

```bash
# After placing data in data/raw/:
python -c "
from src.data_loader import CCLELoader
loader = CCLELoader()
results = loader.load('data/raw/ccle_metabolomics.csv')
loader.print_coverage_report(results)
"
```
