# Tools

Utility scripts for calibration, analysis, and verification. These are NOT part of the core framework — they were used during development.

| Script | Purpose |
|--------|---------|
| `calibrate_optimal.py` | Grid search + local refinement for simulation parameters |
| `ccle_calibration.py` | Calibrate generators against CCLE metabolomics data |
| `combination_sweep.py` | Drug synergy/antagonism pairwise sweep |
| `comprehensive_diagnosis.py` | Legacy diagnostic report generator (superseded by `confluence_runner.py`) |
| `pan_cancer_proof.py` | Original 6-cancer proof script (superseded by `universal_cure_engine.py`) |
| `robustness_analysis.py` | Parameter sensitivity sweep |
| `verify_confluence.py` | Quick verification that the confluence runner works |
